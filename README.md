## Snake Game; Simple

This is a simple Snake game I put together in a short period of time, as an educational piece for C++.
It’s lightweight enough to run on any graphical Linux environment, works fully offline, and keeps things intentionally minimal.

---

## Current Features

**Basic audio (sound effects + music)**

**Grid-based visuals**

**Restart on game over**

***More features are planned.***

---

## Installation:

First download the latest release, wait for that to finish.

Navigate to where it was Downloaded, Then enter the following:

```bash
tar -xJvf SnakeGame-linux.tar.xz
cd Build
./SnakeGame
```
